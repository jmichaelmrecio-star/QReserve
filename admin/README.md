# Admin Folder

This folder contains all the files and resources related to the administration panel of the Tito Renz Resort Web System. The admin panel is designed to allow administrators to manage the resort's operations, including user management, reservations, services, and schedules.

## Files and Directories

- **admin-actions.html**: HTML file for admin actions.
- **admin-component-loader.js**: JavaScript file for dynamically loading admin components.
- **admin-dashboard.css**: Stylesheet for the admin dashboard.
- **admin-dashboard.html**: Main HTML file for the admin dashboard.
- **admin-profile-modal.html**: HTML file for the admin profile modal.
- **admin-promo-codes.html**: HTML file for managing promo codes.
- **admin-reschedule.html**: HTML file for rescheduling reservations.
- **admin-reschedule.js**: JavaScript file for rescheduling logic.
- **admin-reservations.html**: HTML file for managing reservations.
- **admin-reservations.js**: JavaScript file for reservation management.
- **admin-schedule.html**: HTML file for managing schedules.
- **admin-schedule.js**: JavaScript file for schedule management.
- **admin-services.html**: HTML file for managing services.
- **admin-services.js**: JavaScript file for service management.
- **admin-system.js**: Core JavaScript file for admin system functionalities.
- **admin-users.html**: HTML file for managing users.
- **checkin.html**: HTML file for check-in functionalities.
- **payment-verifications.html**: HTML file for verifying payments.
- **payment-verifications.js**: JavaScript file for payment verification logic.

## Usage

1. Open `admin-dashboard.html` in your browser to access the admin dashboard.
2. Use the various pages to manage the resort's operations:
   - **Users**: Manage user accounts and roles.
   - **Reservations**: View, edit, and manage customer reservations.
   - **Services**: Add, update, or remove resort services.
   - **Schedules**: Manage resort schedules and availability.
   - **Promo Codes**: Create and manage promotional codes.
   - **Payment Verifications**: Verify and manage payment records.

## Notes

- Ensure the backend server is running before accessing the admin panel.
- The admin panel is designed for use by authorized personnel only. Ensure proper authentication and authorization are in place.
