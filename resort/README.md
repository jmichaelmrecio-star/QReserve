# Resort Folder

This folder contains JSON files used for testing and development purposes in the Tito Renz Resort Web System. These files simulate the data used by the system, such as user accounts, reservations, services, and more.

## Files and Directories

- **test.account_tbl.json**: Sample data for user accounts.
- **test.blockeddates.json**: Sample data for blocked dates.
- **test.promocodes.json**: Sample data for promotional codes.
- **test.reservations.json**: Sample data for reservations.
- **test.role_tbl.json**: Sample data for user roles.
- **test.services.json**: Sample data for resort services.

## Usage

1. These files are used for testing and development purposes.
2. You can modify these files to simulate different scenarios and test the system's behavior.
3. Ensure that the backend server is running to utilize these test data files.

## Notes

- Do not use these files in a production environment.
- For production, ensure that the database is properly configured and populated with real data.
