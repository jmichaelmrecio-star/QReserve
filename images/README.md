# Images Folder

This folder contains all the image assets used throughout the Tito Renz Resort Norzagaray Reservation System. These images are used for various purposes, including:

## Contents

- **Logo**: Contains the logo of the resort.
- **Room Images**: Images of the different rooms and cottages available for booking.
- **Pool Images**: Photos of the pool area and related amenities.
- **Promotional Banners**: Images used for promotional purposes.
- **Icons**: Small icons used in the user interface.

## Notes

- Ensure that all images are optimized for web usage to improve loading times.
- Use descriptive and consistent naming conventions for image files.
- Store images in appropriate subfolders if necessary to maintain organization.
- Avoid uploading unnecessarily large files to keep the project lightweight.

If you need to add new images, ensure they are properly licensed and follow the project's design guidelines.
