# Feedback Folder

This folder contains files related to the feedback system of the Tito Renz Resort Web System. The feedback system allows customers to provide their feedback about their experience at the resort.

## Files and Directories

- **feedback.html**: HTML file for the feedback form and display.

## Usage

1. Open `feedback.html` in your browser to access the feedback page.
2. Customers can submit their feedback through the form provided on this page.
3. Feedback data is processed and stored in the backend for review by the admin or manager.

## Notes

- Ensure the backend server is running to handle feedback submissions.
- Feedback data is stored in the database and can be accessed via the admin or manager dashboards.
