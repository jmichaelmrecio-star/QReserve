# Manager Folder

This folder contains files related to the manager's dashboard and functionalities in the Tito Renz Resort Web System. The manager dashboard is designed to help managers oversee reservations, schedules, and promotional codes.

## Files and Directories

- **manager-dashboard.html**: Main HTML file for the manager dashboard.
- **manager-promo-codes.html**: HTML file for managing promotional codes.
- **manager-reservations.html**: HTML file for managing reservations.
- **manager-schedule.html**: HTML file for managing schedules.

## Usage

1. Open `manager-dashboard.html` in your browser to access the manager dashboard.
2. Use the various pages to perform the following tasks:
   - **Promo Codes**: Manage promotional codes for discounts and offers.
   - **Reservations**: View and manage customer reservations.
   - **Schedules**: Oversee and manage resort schedules.

## Notes

- Ensure the backend server is running to fetch and update data.
- The manager dashboard is designed for use by authorized personnel only. Ensure proper authentication and authorization are in place.
